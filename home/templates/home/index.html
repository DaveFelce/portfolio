Estate agent demo

A Django app that allows estate agents to list properties for rent and sale. Properties can be searched for by users entering location and description keywords and they can also register to receive a regular email based on past search criteria.

The properties are stored in a Postgresql database, which regularly populates an Elasticsearch index. Elasticsearch is a NoSQL storage & search engine.

Try searching for properties, for example 'London' for location and 'spacious luxurious' as the keywords in the property description.

What's in my fridge demo

A Django app that lets you enter ingredients found in your cupboards or fridge, and suggests recipes that best match what you have entered.

Elasticsearch is again used as the search engine, although recipes are entered through the Django Rest Framework powered API and live in a Postgresql database.

Type in a few ingredients and obseve the beautiful charts! If you hover over the recipe names on the left, you will see a popup showing the ingredient list and which ones you matched.
